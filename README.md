## Description

Project to provide informations of about countries.

## 

Clone the project

```bash
    git clone git@github.com:lfeitosaf/country-info-back-end.git
```

Go to the directory

```bash
 cd country-info-back-end
```

Install the dependencies

```bash
  npm install
```

Run the app

```bash
  npm run start:dev
```
